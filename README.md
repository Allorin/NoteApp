# NoteListApp

Простое и удобное приложение для управления заметками, разработанное на современном стеке Android-технологий.

## О приложении

Приложение позволяет пользователю создавать, редактировать и удалять заметки.
Заметки можно отмечать как выполненные (с зачеркиванием текста) и сортировать по категориям.

### Основной функционал:
- Создание, редактирование и удаление заметок.
- Категории (создание своих категорий или выбор из списка).
- Отметка выполненных задач (визуальное изменение стиля).
- Сохранение данных между сессиями (Локальная БД).
- Адаптация под смену ориентации экрана.

## Технологии

Проект разработан с использованием:
- **Kotlin**
- **Jetpack Compose** (Современный UI)
- **Material 3**
- **Room Database** (Локальное хранение данных)
- **MVVM** (Архитектура)
- **Coroutines & Flow** (Асинхронность)

## Установка

1. Скачайте APK файл из раздела **Releases**.
2. Установите на устройство (Android 7.0+).
